this is initial version - 1.0

Install latest python 
add python to the PATH

Run install.bat & wait till it finishes 
then run the application from cmd prompt : python DayOpenInterest.py
or
just click run.bat to launch application 


Note1 : wait some time to accumulate data so that you will see charts
Note2 : if you don't see any "Expiry List" in the UI for long time, restart the application  
Note3 : if you see any issues in the application check log created in the current directory or exception messages on console
